A simple command-line calculator built using Python. This program allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

 Features
.Add two numbers
.Subtract two numbers
.Multiply two numbers
.Divide two numbers
.Handles division by zero errors
.Handles invalid input (e.g., letters instead of numbers)
.Continuous operation until the user chooses to exit

 How It Works
.The program displays a menu of operations.
.The user selects an option (1–5).
.The user enters two numbers.
.The program performs the selected operation and displays the result.
.The program repeats until the user selects Exit (5).
